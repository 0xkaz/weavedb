# Clone the Repo

[WeaveDB](https://github.com/asteroid-dao/weavedb) is a monorepo that consists of Warp contracts, SDK, REPL, a web console, and many helper scripts.

```bash
git clone https://github.com/asteroid-dao/weavedb.git
cd weavedb
yarn
```

Test.

```bash
yarn test
```
